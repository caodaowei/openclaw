### 问题2：飞书插件路径错误
**现象**: 启动时显示 `plugin path not found: /home/azureuser/.nvm/versions/node/v24.14.0/lib/node_modules/openclaw/extensions/feishu`
**原因**: feishu插件在配置中启用但未安装

**解决**:
```bash

```
