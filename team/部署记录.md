## 🔧 Phase 3: 环境配置

### 3.1 SSH 免密登录配置

**配置步骤**:
1. 确认本地已有 SSH 密钥
2. 将公钥添加到腾讯OpenClaw的 `~/.ssh/authorized_keys`
3. 创建 `~/.ssh/config` 配置主机别名

**配置内容**:
```
Host 腾讯OpenClaw
    HostName 43.156.51.119
    User root
    Port 22
    IdentityFile ~/.ssh/txCloud_key.pem
```

**测试命令**:
```bash
ssh 腾讯OpenClaw "hostname"
# 输出: VM-0-9-opencloudos
```
