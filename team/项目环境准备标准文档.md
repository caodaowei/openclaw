## 2. 新项目启动清单

### 2.1 AITest01 准备（运维）

```bash
# 1. 创建项目数据库
sudo docker run -d \
  --name {项目名}-postgres \
  -e POSTGRES_USER={项目名} \
  -e POSTGRES_PASSWORD={密码} \
  -e POSTGRES_DB={项目名} \
  -v ~/{项目名}-data/postgres:/var/lib/postgresql/data \
  -p 5432:5432 \
  postgres:15-alpine

# 2. 配置防火墙白名单（仅允许腾讯OpenClaw访问PostgreSQL）
sudo ufw allow from 43.156.51.119 to any port 5432 comment '{项目名} PostgreSQL'
sudo ufw allow 22 comment 'SSH'
sudo ufw allow 80 comment 'HTTP'
sudo ufw allow 443 comment 'HTTPS'
sudo ufw --force enable

# 3. 验证连接
sudo docker exec {项目名}-postgres pg_isready -U {项目名}
```

### 2.2 腾讯OpenClaw 准备（开发）

```bash
# 1. 创建项目目录
mkdir -p ~/src/{项目名}
cd ~/src/{项目名}
git init

# 2. 初始化后端项目
mkdir -p backend/src/{controllers,services,models,routes,middleware,utils}
cat > backend/package.json << 'EOF'
{
  "name": "{项目名}-backend",
  "version": "1.0.0",
  "type": "module",
  "scripts": {
    "dev": "tsx watch src/index.ts",
    "build": "tsc",
    "start": "node dist/index.js"
  },
  "dependencies": {
    "express": "^4.18.2",
    "cors": "^2.8.5",
    "pg": "^8.11.3",
    "dotenv": "^16.3.1"
  },
  "devDependencies": {
    "@types/express": "^4.17.21",
    "@types/cors": "^2.8.17",
    "@types/pg": "^8.10.9",
    "@types/node": "^20.10.0",
    "typescript": "^5.3.2",
    "tsx": "^4.6.0"
  }
}
EOF

# 3. 配置环境变量
cat > backend/.env << 'EOF'
DATABASE_URL=postgresql://{用户名}:{密码}@43.156.51.119:5432/{数据库名}
PORT=3000
NODE_ENV=development
EOF

# 4. 创建入口文件
cat > backend/src/index.ts << 'EOF'
import express from 'express';
import cors from 'cors';
import dotenv from 'dotenv';

dotenv.config();

const app = express();
const PORT = process.env.PORT || 3000;

app.use(cors());
app.use(express.json());

app.get('/api/health', (req, res) => {
  res.json({ status: 'ok', time: new Date().toISOString() });
});

app.listen(PORT, () => {
  console.log(`🚀 Server running on port ${PORT}`);
});
EOF

# 5. 提交到 Git
git add -A
git commit -m "初始化项目"
```

## 3. DevOps 流程配置

### 3.1 GitHub Actions CI/CD

创建 `.github/workflows/ci.yml`:

```yaml
name: CI/CD

on:
  push:
    branches: [ main, develop ]
  pull_request:
    branches: [ main ]

jobs:
  test:
    runs-on: ubuntu-latest
    
    services:
      postgres:
        image: postgres:15-alpine
        env:
          POSTGRES_USER: {项目名}
          POSTGRES_PASSWORD: {密码}
          POSTGRES_DB: {项目名}_test
        options: >-
          --health-cmd pg_isready
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
        ports:
          - 5432:5432

    steps:
    - uses: actions/checkout@v4
    
    - name: Setup Node.js
      uses: actions/setup-node@v4
      with:
        node-version: '20'
    
    - name: Install dependencies
      working-directory: ./backend
      run: npm ci
    
    - name: Run tests
      working-directory: ./backend
      env:
        DATABASE_URL: postgresql://{项目名}:{密码}@localhost:5432/{项目名}_test
      run: npm test

  deploy:
    needs: test
    runs-on: ubuntu-latest
    if: github.ref == 'refs/heads/main'
    
    steps:
    - uses: actions/checkout@v4
    
    - name: Deploy to AITest01
      uses: appleboy/ssh-action@master
      with:
        host: ${{ secrets.AITEST01_HOST }}
        username: ${{ secrets.AITEST01_USER }}
        key: ${{ secrets.AITEST01_KEY }}
        script: |
          cd ~/src/{项目名}
          git pull origin main
          docker-compose -f docker-compose.prod.yml down
          docker-compose -f docker-compose.prod.yml up -d --build
```

### 3.2 生产 Docker 配置

创建 `docker-compose.prod.yml`:

```yaml
version: '3.8'

services:
  backend:
    build:
      context: ./backend
      dockerfile: Dockerfile
    environment:
      - NODE_ENV=production
      - DATABASE_URL=postgresql://{用户名}:${DB_PASSWORD}@postgres:5432/{数据库名}
      - PORT=3000
    depends_on:
      - postgres
    restart: unless-stopped

  frontend:
    build:
      context: ./frontend
      dockerfile: Dockerfile
    ports:
      - "80:80"
    depends_on:
      - backend
    restart: unless-stopped

  # 使用外部 PostgreSQL 或内部
  # postgres:
  #   image: postgres:15-alpine
  #   ...
```

### 3.3 后端 Dockerfile

创建 `backend/Dockerfile`:

```dockerfile
# 构建阶段
FROM node:20-alpine AS builder

WORKDIR /app

COPY package*.json ./
RUN npm ci

COPY . .
RUN npm run build

# 生产阶段
FROM node:20-alpine

WORKDIR /app

COPY package*.json ./
RUN npm ci --only=production

COPY --from=builder /app/dist ./dist

EXPOSE 3000

USER node

CMD ["node", "dist/index.js"]
```

## 8. 安全加固

### 8.1 数据库安全

#### 防火墙白名单（推荐）

限制 PostgreSQL 仅允许特定 IP 访问：

```bash
# 允许腾讯OpenClaw访问PostgreSQL
sudo ufw allow from 43.156.51.119 to any port 5432

# 允许必要服务
sudo ufw allow 22      # SSH
sudo ufw allow 80      # HTTP
sudo ufw allow 443     # HTTPS

# 启用防火墙
sudo ufw enable

# 查看状态
sudo ufw status verbose
```

#### PostgreSQL 自带白名单（可选）

修改 `pg_hba.conf`：

```conf
# 允许特定IP
host    {数据库名}    {用户名}    43.156.51.119/32    scram-sha-256

# 拒绝其他
host    all           all         0.0.0.0/0           reject
```

### 8.2 密码安全

- 使用强密码（12位以上，包含大小写+数字+特殊字符）
- 定期更换密码
- 不在代码中硬编码密码，使用环境变量

### 8.3 网络安全

- 数据库不直接暴露公网（通过防火墙限制）
- 使用 SSH 隧道或 VPN 访问数据库
- 定期更新系统和软件补丁
