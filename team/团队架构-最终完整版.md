## 🎯 对接机制

### 大哥 ↔ 小白（唯一对接人）

**小白是唯一与大哥直接对接的角色**

```
大哥
  │
  ▼
小白（项目经理）
  │
  ├─ 分析需求
  ├─ 分配任务给团队
  ├─ 跟踪进度
  ├─ 汇总结果
  │
  ▼
汇报大哥
```

**小白的职责**:
- 接收大哥所有指令
- 分析并拆解任务
- 分配给团队成员
- 跟踪执行进度
- 汇总结果汇报
- 处理异常情况

**其他成员不直接与大哥交互**，所有沟通通过小白中转。

## 🚀 快速部署脚本

### AITest01 部署（4个Agent）

```bash
#!/bin/bash
# deploy-aitest01.sh

set -e
echo "🚀 部署 AITest01 四个 Agent..."

# 创建目录
mkdir -p ~/.openclaw/workspace/agents/{小白-项目经理,小红-产品经理,小黑-测试专员,小青-运维专员}
mkdir -p ~/.openclaw/agents/{xiaobai-agent,xiaohong-agent,xiaohei-agent,xiaoqing-agent}/agent

# 创建身份文件（略，见下方身份文件模板）

# 备份配置
cp ~/.openclaw/openclaw.json ~/.openclaw/openclaw.json.bak.$(date +%Y%m%d_%H%M%S)

# 添加Agent配置
python3 << 'PYEOF'
import json
with open("/home/azureuser/.openclaw/openclaw.json", "r") as f:
    config = json.load(f)

if "agents" not in config:
    config["agents"] = {}
if "list" not in config["agents"]:
    config["agents"]["list"] = []

agents = [
    {"id": "xiaobai-agent", "name": "小白-项目经理", "workspace": "/home/azureuser/.openclaw/workspace/agents/小白-项目经理", "agentDir": "/home/azureuser/.openclaw/agents/xiaobai-agent", "identity": {"name": "小白"}},
    {"id": "xiaohong-agent", "name": "小红-产品经理", "workspace": "/home/azureuser/.openclaw/workspace/agents/小红-产品经理", "agentDir": "/home/azureuser/.openclaw/agents/xiaohong-agent", "identity": {"name": "小红"}},
    {"id": "xiaohei-agent", "name": "小黑-测试专员", "workspace": "/home/azureuser/.openclaw/workspace/agents/小黑-测试专员", "agentDir": "/home/azureuser/.openclaw/agents/xiaohei-agent", "identity": {"name": "小黑"}},
    {"id": "xiaoqing-agent", "name": "小青-运维专员", "workspace": "/home/azureuser/.openclaw/workspace/agents/小青-运维专员", "agentDir": "/home/azureuser/.openclaw/agents/xiaoqing-agent", "identity": {"name": "小青"}}
]

existing = [a.get("id") for a in config["agents"]["list"]]
for agent in agents:
    if agent["id"] not in existing:
        config["agents"]["list"].append(agent)
        print(f"✅ 添加: {agent['name']}")

with open("/home/azureuser/.openclaw/openclaw.json", "w") as f:
    json.dump(config, f, indent=2, ensure_ascii=False)
print("🎉 AITest01 部署完成！")
PYEOF

# 重启
openclaw gateway restart
```
