# AITest01 端口规划文档

> 本机所有服务的端口分配规划

## 端口分配原则

| 范围 | 用途 | 说明 |
|------|------|------|
| 5001-5999 | 本地开发 | 开发环境使用 |
| 7001-7799 | 自研生产应用 | 正式部署的自研应用服务 |
| 7801-7999 | 非自研服务 | 数据库、第三方开源应用、基础设施等 |

---

## 端口冲突处理

如遇端口冲突，检查步骤：

```bash
# 查看端口占用
sudo netstat -tlnp | grep :7001

# 查看 Docker 容器
sudo docker ps -a

# 停止冲突服务
sudo docker stop <container_name>
```

## 新增服务端口申请流程

1. **自研应用** (7001-7799)
   - 从 7001 开始顺序分配
   - 在本文档中登记

2. **数据库/第三方开源** (7801-7999)
   - 从 7801 开始顺序分配
   - 在本文档中登记并标注类型

3. **本地开发** (5001-5999)
   - 与生产端口对应，便于记忆
   - Backend: 5001, Frontend: 5002
