| **Docker** | 20.10+ |
| **Docker Compose** | 2.0+ |

---

## 2. 部署步骤

### 2.1 克隆代码

```bash
cd /home/azureuser/src
git clone https://github.com/caodaowei/foxchat.git
cd foxchat
```

### 2.2 启动服务

```bash
# 启动所有服务
docker-compose up -d

# 查看状态
docker-compose ps

# 查看日志
docker-compose logs -f
```

### 2.3 初始化数据库

```bash
# 执行数据库迁移
cd backend
npm install
npm run db:migrate
```

### 2.4 验证部署

```bash
# 检查前端
curl http://localhost

# 检查后端API
curl http://localhost:3000/api/health

# 检查数据库
docker-compose exec postgres pg_isready -U foxchat
```

## 3. 配置说明

### 3.1 环境变量

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| `DATABASE_URL` | 数据库连接 | postgresql://foxchat:foxchat123@postgres:5432/foxchat |
| `PORT` | 后端端口 | 3000 |
| `FEISHU_WEBHOOK_URL` | 飞书Webhook | - |

### 3.2 修改配置

```bash
# 编辑环境变量
vim backend/.env

# 重启服务
docker-compose restart backend
```

## 4. 备份策略

### 4.1 数据库备份

```bash
# 手动备份
docker-compose exec postgres pg_dump -U foxchat foxchat > backup_$(date +%Y%m%d).sql

# 自动备份（添加到crontab）
0 2 * * * cd /home/azureuser/src/foxchat && docker-compose exec -T postgres pg_dump -U foxchat foxchat > /backup/foxchat_$(date +\%Y\%m\%d).sql
```

### 4.2 数据恢复

```bash
# 恢复数据
docker-compose exec -T postgres psql -U foxchat < backup_20260319.sql
```

## 6. 故障排查

### 6.1 服务无法启动

```bash
# 检查端口占用
netstat -tlnp | grep -E '80|3000|5432'

# 检查Docker状态
systemctl status docker
```

### 6.2 数据库连接失败

```bash
# 检查Postgres状态
docker-compose ps postgres

# 查看Postgres日志
docker-compose logs postgres
```
