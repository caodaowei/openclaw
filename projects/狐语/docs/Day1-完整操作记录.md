## 三、开发环境部署（14:00-16:00）

### 3.1 AITest01环境部署

#### PostgreSQL部署
```bash
# 安装Docker
curl -fsSL https://get.docker.com | sh

# 部署PostgreSQL
sudo docker run -d \
  --name openclaw-postgres \
  -e POSTGRES_USER=foxchat \
  -e POSTGRES_PASSWORD=foxchat123 \
  -e POSTGRES_DB=foxchat \
  -v ~/openclaw-data/postgres:/var/lib/postgresql/data \
  -p 5432:5432 \
  postgres:15-alpine
```

**问题3**：Docker权限问题
- 安装后需要重新登录或sudo
- 解决：使用sudo运行docker命令

#### 防火墙配置
```bash
# 配置白名单，仅允许腾讯OpenClaw访问
sudo ufw allow from 43.156.51.119 to any port 5432
sudo ufw allow 22
sudo ufw allow 80
sudo ufw allow 443
sudo ufw enable
```
