## 文档规范

### 命名规范

- 需求文档：`需求文档-v{版本}.md`
- 设计文档：`{类型}设计.md`
- 开发日志：`{日期}-{主题}.md`
- 会议记录：`{类型}-{日期}.md`

### 更新规范

- 需求变更：更新版本号，保留历史
- 开发日志：每日更新，记录进度
- 会议记录：会后24小时内完成
