# 狐语 FoxChat 核心需求（最终版）

> OpenClaw 生态沟通平台
> 所有项目团队，一个沟通入口

---

## 核心定位

**狐语 = OpenClaw 生态的统一沟通平台**

- 不只是 FoxChat 项目的沟通工具
- 而是 **所有 OpenClaw 项目团队的沟通入口**
- 支持团队持续扩展，新团队随时加入

---

## 核心需求

### 1. 多团队支持（核心）

**场景**：
```
狐语平台
├── FoxChat 团队（当前）
│   ├── 小白
│   ├── 小红
│   ├── 小灰
│   ├── 小蓝
│   ├── 小黑
│   └── 小青
│
├── iQuant.Portal 团队（未来）
│   ├── 项目经理
│   ├── 开发
│   └── 测试
│
├── Learning 团队（未来）
│   └── 学习者
│
└── 新团队X（随时加入）
    └── ...
```

**功能**：
- 每个项目有独立沟通空间
- 团队成员按项目组织
- 支持团队间协作（可选）

### 2. 成员管理（核心）

**读取所有项目成员**：
- 自动同步各项目成员列表
- 成员角色、状态一目了然
- 支持跨项目查看成员

**成员加入流程**：
```
新项目成立
  ↓
在狐语创建团队
  ↓
添加团队成员
  ↓
成员激活，开始沟通
```

### 3. 持续扩展（核心）

**支持无限扩展**：
- 团队数量无上限
- 成员数量无上限
- 新团队随时加入
- 老团队随时归档

## 架构调整

### 数据模型

```typescript
// 团队表
interface Team {
  id: string;
  name: string;           // 团队名称：FoxChat / iQuant.Portal
  description: string;
  createdAt: string;
  members: User[];        // 团队成员
}

// 沟通记录表
interface Communication {
  id: string;
  teamId: string;         // 关联团队
  title: string;
  type: CommType;
  participants: string[]; // 参与人
  content: string;
  createdAt: string;
}

// 用户表（跨团队）
interface User {
  id: string;
  name: string;
  teams: string[];        // 所属团队列表
  role: UserRole;
  status: UserStatus;     // 活跃/休眠/离线
}
```

### 界面调整

**首页**：
- 团队列表（FoxChat / iQuant.Portal / ...）
- 快速切换团队
- 最近沟通预览

**团队内页**：
- 该团队的沟通记录
- 该团队的成员状态
- 创建新沟通

**成员总览页**：
- 所有项目成员列表
- 按团队筛选
- 成员状态统计
